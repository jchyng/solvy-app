# 00. 제품 정체성 · 타깃 · 차별화

> **이 문서는 모든 주차에서 참조됩니다.** 제품의 "왜"에 해당하는 내용.

---

## 1. 한 줄 정의

**Solvy**는 학생이 수학 문제 사진을 올리면 AI가 **출제 의도와 활용 개념을 분석하고, 최적의 풀이법과 실전 팁을 제공**하는 모바일 웹 서비스입니다. 단순히 답만 알려주는 기존 경쟁사(콴다)와 달리, **"학생 개인의 풀이 노트가 되는 앱"**을 지향합니다.

### 태그라인

> "찍으면 풀이가 나오고, 쌓이면 나의 노트가 됩니다."

대체 후보: "답이 아니라, 풀이법을 가르칩니다." / "내가 푼 모든 문제, 하나의 노트로."

---

## 2. 타깃 사용자

**Phase 1 타깃**: **중·고등학생 사용자 (본인 결제)**

이유:
- 콴다의 광고·AI 오답에 지친 상위권·중위권 학생이 1차 타깃
- "내가 푼 문제로 나만의 노트를 만든다"는 자기 주도 학습 서사와 일치
- 풀이법과 실전 팁의 품질 차이를 **본인이 직접 체감**하는 세그먼트

**타깃이 아닌 것**:
- 단순히 답만 빨리 알고 싶은 학생 (콴다로 남겨둠)
- 학부모 직접 결제 유저 (Phase 1 범위 밖, 대시보드·리포트 기능 없음)
- 학원 교사 B2B (Phase 3에서 고려)

### 비전 요약

1. **Phase 1 (현재~6개월)**: 중·고등학생 직접 결제 시장 진입, "풀이 노트" 기능으로 리텐션 확보
2. **Phase 2 (6~18개월)**: 상위권·경시 세그먼트 유입, 과목 확장 (물리·화학)
3. **Phase 3 (18개월+)**: B2B(학원·과외), 해외 진출 검토

## 3. 핵심 차별화: "풀이 노트"

다른 축(정확도, 학습 루틴, 해외 진출)보다 **"풀이 노트"가 제품의 제1 축**입니다. 모든 설계 결정은 이 축을 기준으로 판단합니다.

콴다는 "답을 빨리 얻는 도구"입니다. Solvy는 **"문제 풀이가 학생 개인 자산으로 쌓이는 도구"**입니다.

### 3.1 문제 분석 + 최적 풀이 + 실전 팁 — 제품의 심장

학생이 문제 사진을 올리면 AI는 **한 번에** 다음을 제공합니다:

- **출제 의도** — 이 문제는 무엇을 테스트하려는가
- **활용 개념** — 어떤 단원·공식이 쓰이는가
- **최적 풀이** — 가장 효율적인 단계별 방법
- **실전 팁** — 시험장에서 시간 단축·실수 회피법

다른 풀이법이 있을 경우 본문에 쏟아붓지 않고 **꼬리 질문 칩**으로 유도:

> "Q1. 다른 풀이 방법도 알려줘?"
> "Q2. 비슷한 유형 하나만 만들어줘"
> "Q3. 이 개념 처음인데 먼저 뭘 공부해?"

학생이 탭하면 **동일한 대화방**에서 후속 응답이 이어집니다.

### 3.2 대화형 후속 질문 (멀티턴)

문제 하나당 **하나의 대화방(채팅)**. 꼬리 질문뿐 아니라 **자유 입력**으로 계속 질문 가능:

- "2번 단계에서 왜 양변을 제곱해?"
- "이거 몇 학년 몇 학기 내용이야?"
- "비슷한 난이도 문제 하나 만들어줘"

대화 히스토리는 전부 저장되고, 이후 노트로 보관될 때 함께 보존됩니다.

### 3.3 풀이 노트 (즐겨찾기 + 목록 + 이름 지정)

**핵심 인사이트**: 학생이 "이 풀이는 나중에 다시 보고 싶다"고 느끼는 순간이 있습니다. 이 순간을 놓치지 않고 **개인 자산화**하는 것이 Solvy의 차별점입니다.

- 모든 대화는 기본적으로 **일반 채팅 목록**에 저장 (최근순)
- **★ 즐겨찾기**
- 사용자가 만든 **목록(폴더)**으로 분류 — 예: "이차함수 킬러", "수능 기출 오답"
- **이름 직접 지정** 가능 — 예: "2024 수능 30번"
- 한 풀이는 여러 목록에 중복 소속 가능 (태그 개념)

**결국 앱 자체가 학생의 풀이 노트가 됩니다.** 콴다가 제공하지 않는 영속성·소유감이 리텐션 무기입니다.

## 4. 벤치마크: 콴다(QANDA)

리뷰 75.5만개, 평점 4.0. 장점과 불만 모두 Solvy 설계의 레퍼런스입니다.

### 참고할 강점
- 사진 한 장으로 5초 만에 풀이 (최고 만족 요인)
- 2300만개 문제 DB, 한국 교육과정 특화
- 유사 문제 추천
- 1:1 선생님 풀이 + 동영상 (AI+사람 하이브리드)

### 반복하면 안 될 불만
1. **광고 범람** — 문제 1개 검색에 2~3개 광고
2. **AI 정확도 낮음** — "5개 찍으면 3개 틀린다"
3. **무료 → 유료 기습 전환**
4. **기본값 강제 변경** — 업데이트 후 사용자 동선 변경
5. **OCR 오인식 증가**
6. **템플릿성 고객 대응**
7. **풀이 보존·관리 부재** — 푼 문제가 개인 자산이 되지 않음
