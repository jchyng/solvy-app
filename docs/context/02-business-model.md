## 베타 중: 무료, 측정 중심

**8주 베타 = 데이터 수집 기간**. 가격을 미리 확정하지 않고 베타 데이터로 결정합니다.

### 수집할 데이터

1. **사용자당 실제 API 비용** (헤비/미디엄/라이트 분포, 채팅 턴 수 포함)
2. **기능별 사용 빈도** (문제 업로드 vs 후속 대화 vs 노트 즐겨찾기·목록 생성)
3. **리텐션 상관 변수** — 노트에 저장하는 학생은 남는가? 목록을 만든 학생은 남는가?
4. **광고 시청 의향 및 실제 eCPM**

## 예상 구독 플랜 (베타 후 확정)

| 티어 | 월 가격 | 포함 사용량 | 타깃 |
|---|---|---|---|
| Free | 0원 | 하루 3문제, 노트 최대 20개 | 체험용 |

가격 공식: `월 가격 = (플랜별 평균 API 비용 × 2.5) + 운영비 몫`

## 광고 리워드 시스템

**원칙**: 강제 광고는 없다. 단, 자발적으로 광고 보고 크레딧 받는 옵션은 제공.

- **대상**: Free + Light 티어만. Pro는 광고 옵션 자체를 숨김
- **조건**:
  - 교육·학습용품·도서·문구만 (광고주 카테고리 화이트리스트)
  - 15초 이내 스킵 가능한 리워드 광고만
  - 인터랙티브·미니게임형 광고 절대 금지
  - "오늘 광고 없이 보기" 옵션 (당일 권한 구매 300~500원)
- **환율**: 광고 1편 = 1 크레딧 (베타 후 조정)
- **상한**: 일일 5 크레딧 (남용 방지)

## 베타 참여자 혜택 (반드시 약속 이행)

콴다의 "무료였던 게 유료가 됐다" 실수를 반복하지 않기 위해:

- 정식 출시 시 **가격 50% 할인 첫 12개월**
- Pro 티어 **평생 20% 할인 lock-in** (해지 전까지)
- 출시 버전 크레딧 30일치 선지급
- "Solvy Founding Member" 뱃지
- **베타 중 만든 노트·목록은 정식 출시 후에도 영구 보존 보장**

이 약속은 문서로 명문화하고, 베타 가입 시 사용자에게 명시.

## 정식 출시 후 수익화 원칙

1. 무료로 시작한 기능은 절대 유료화하지 않는다
2. 기본값 변경 금지 (사용자 동선 강제 변경 안 함)
3. AI 정확도 불안정하면 AI-only로 밀지 않음 (인식 확인 단계 유지)
4. 광고는 도입하지 않음 (리워드 방식만)
5. 사용자 노트 데이터는 절대 학습에 쓰지 않는다 (개인정보처리방침 명시)
