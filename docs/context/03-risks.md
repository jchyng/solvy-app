## 기술적 위험

| 위험 | 확률 | 영향 | 대응 |
|---|---|---|---|
| Claude Sonnet 장애 | 중 | 크리티컬 | DeepSeek R1, Gemini Pro 폴백 체인 필수 |
| 대화 히스토리 비용 폭발 | 중 | 높음 | 프롬프트 캐싱 + 오래된 턴 요약 압축 |
| 분석 품질 저하 | 중 | 크리티컬 | 벤치마크 자동 회귀, 임계치 하락 시 알림 |
| 꼬리 질문 빈 칩 또는 반복 | 중 | 중 | 프롬프트에 다양성·중복 방지 규칙, 내부 평가 |
| AI 모델 버전 폐기 | 낮음 | 중 | 모델명 config, 폴백 체인 |
| 사용자 폭증으로 비용 폭발 | 낮음 | 높음 | 일일 비용 알림, 자동 레이트리밋 |
| OCR 오인식률 상승 | 중 | 중 | confidence 임계치로 확인 단계 자동 진입 |

## 비즈니스 위험

| 위험 | 대응 |
|---|---|
| 학생이 노트화 기능을 쓰지 않음 | 온보딩에서 "즐겨찾기 1회" 경험 필수 포함, 1주 차 리마인드 |
| 단순 답 조회 용도로만 쓰고 이탈 | 분석 품질 차별화 유지, 꼬리 질문으로 대화 유도 |
| 콴다가 유사 기능 출시 | 품질·정직성·노트 영속성 차별화 유지 |
| 초기 AI 오답으로 신뢰 상실 | confidence 낮으면 "확실하지 않다" 표시, 신고 즉시 대응 |

## 보안·개인정보 위험

- **API 키 누설** — 프론트 노출 금지, gitleaks 스캔, Cloudflare Secrets 사용
- **사용자 간 데이터 교차 접근** — 모든 엔드포인트에 소유권 체크 미들웨어 (타인 `conversation_id`, `folder_id`, `problem_session_id` 차단)
- **만 14세 미만 보호자 동의** — 한국 개인정보보호법 준수 (학부모 계정 없이 동의만 수집)
- **노트 데이터 학습 사용 금지** — 개인정보처리방침에 명시

→ 구체적 체크리스트는 `sprints/week-7-hardening.md`에 있습니다.
